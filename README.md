# Frontend TypeScript Setup

Minimal frontend project with:

- `frontend/`: React + TypeScript + Vite + Tailwind CSS

Firebase is not configured yet.

## Prerequisites

- Node.js
- npm

If PowerShell blocks `npm`, use `npm.cmd` instead.

## Install

```powershell
cd frontend
npm install
```

## Run

### Development server

```powershell
cd frontend
npm run dev
```

### Production build

```powershell
cd frontend
npm run build
```

## Notes

- The frontend contains one simple landing page only.
- Firebase can be added later in `frontend/` when you are ready.
